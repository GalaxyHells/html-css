Principios da formatação

Negrito / Destaque

Essa é uma frase em Negrito usando a tag <B> (não Semantica)

Essa é uma frase negrito usando a tag <strong> (Semantica)

Italico / Ênfase

Essa frase esta em Italico usando a tag <I> (Não semantica)

Essa frase esta em italico usando a tag <em> (Semantica)

Texto Marcado

Este é um texto marcado utilizando <mark>

Para trocar a cor do marcado em um ponto especifico utilize <style>

Deve criar uma tag Style no header pra mudar a cor

Texto Pequeno e Grande

A tag small pode ser utilizada, ja big não

Texto Excluido

Podemos marcar um texto excluido para que ele possa ser lido mas não considerado usando a tag <del> (Semantico)

Texto Inserido

Podemos marcar um texto como inserido para dar enfase ao texto usando a tag <ins>(Semantico)

Existe também o texto sublinhado com a tag <u> (Não semantico)

Texto sobrescrito

Para inserir coisas como x20+3 usando a tag <sup>

Texto subscrito

Para inserir coisas como H2O usando a tag <sub>
